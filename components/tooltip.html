Tooltip 弹出气泡 ​

常用的 Tooltip

基础用法 ​

基础 Tooltip，通过 content 设置气泡内的内容，placement 设置气泡的位置，触发元素的内容以插槽的形式传入。

<template>
  <div class="tooltip-base-box">
    <div class="row center">
      <my-tooltip
        class="box-item"
        content="Top Left prompts info"
        placement="top-start"
      >
        <my-button>top-start</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Top Center prompts info"
        placement="top"
      >
        <my-button>top</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Top Right prompts info"
        placement="top-end"
      >
        <my-button>top-end</my-button>
      </my-tooltip>
    </div>
    <div class="row">
      <my-tooltip
        class="box-item"
        content="Left Top prompts info"
        placement="left-start"
      >
        <my-button>left-start</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Right Top prompts info"
        placement="right-start"
      >
        <my-button>right-start</my-button>
      </my-tooltip>
    </div>
    <div class="row">
      <my-tooltip
        class="box-item"
        content="Left Center prompts info"
        placement="left"
      >
        <my-button class="mt-3 mb-3">left</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Right Center prompts info"
        placement="right"
      >
        <my-button>right</my-button>
      </my-tooltip>
    </div>
    <div class="row">
      <my-tooltip
        class="box-item"
        content="Left Bottom prompts info"
        placement="left-end"
      >
        <my-button>left-end</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Right Bottom prompts info"
        placement="right-end"
      >
        <my-button>right-end</my-button>
      </my-tooltip>
    </div>
    <div class="row center">
      <my-tooltip
        class="box-item"
        content="Bottom Left prompts info"
        placement="bottom-start"
      >
        <my-button>bottom-start</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Bottom Center prompts info"
        placement="bottom"
      >
        <my-button>bottom</my-button>
      </my-tooltip>
      <my-tooltip
        class="box-item"
        content="Bottom Right prompts info"
        placement="bottom-end"
      >
        <my-button>bottom-end</my-button>
      </my-tooltip>
    </div>
  </div>
</template>

禁用状态 ​

你可以使用 disabled 属性来定义 Tooltip 是否被禁用。

<template>
  <div class="wrapper">
    <my-tooltip class="my-tooltip" placement="top" content="no" disabled>
      <my-button>Disabled</my-button>
    </my-tooltip>
    <my-tooltip class="my-tooltip" placement="top-start" content="no diabled">
      <my-button>Disabled</my-button>
    </my-tooltip>
  </div>
</template>

<style>
.wrapper {
  height: 300px;
}
.my-tooltip {
  width: 110px;
  margin-right: 20px;
  margin-top: 20px;
}
</style>

触发条件 ​

使用 trigger属性来定义触发条件，可以为click hover 和 contextmenu

<template>
  <div class="wrapper">
    <my-tooltip
      class="my-tooltip"
      placement="top-start"
      content="hover"
      trigger="hover"
    >
      <my-button>hover</my-button>
    </my-tooltip>
    <my-tooltip
      class="my-tooltip"
      placement="top-start"
      content="click"
      trigger="click"
    >
      <my-button>click</my-button>
    </my-tooltip>
    <my-tooltip
      class="my-tooltip"
      placement="top-start"
      content="contextmenu"
      trigger="contextmenu"
    >
      <my-button>contextmenu</my-button>
    </my-tooltip>
  </div>
</template>

<style>
.wrapper {
  height: 300px;
}
.my-tooltip {
  width: 110px;
  margin-right: 20px;
  margin-top: 20px;
}
</style>

自定义 content ​

使用具名插槽来实现自定义 content

<template>
  <div class="wrapper">
    <my-tooltip class="my-tooltip" placement="top-start">
      <my-button>Content Slot</my-button>
      <template #content>
        <h3>hihi!</h3>
        <p>This is me</p>
      </template>
    </my-tooltip>
  </div>
</template>

<style>
.wrapper {
  height: 300px;
}
.my-tooltip {
  width: 110px;
  margin-right: 20px;
  margin-top: 20px;
}
</style>
