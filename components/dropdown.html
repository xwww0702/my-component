Dropdown 下拉菜单 ​

常用的 Dropdown

基础用法 ​

基础 Dropdown. item 设置下拉框的内容，item 的类型是DropdownItemProps[]，也可以通过 slot 传入

slot 传入 item
通过 props 传入 item

<script setup lang="ts">
import { type DropdownItemProps } from "@qiwen72/my-component";

const items: DropdownItemProps[] = [
  { command: "1", label: "Item 1" },
  { command: "2", label: "Item 2" },
  { command: "3", label: "Item 3", disabled: true },
  { command: "4", label: "Item 4", divided: true },
];
</script>

<template>
  <div class="row">
    <div class="col">
      <div class="desc">slot 传入 item</div>
      <my-dropdown>
        <span class="dropdown-link">
          Dropdown List
          <my-icon icon="angle-down" />
        </span>
        <template #dropdown>
          <my-dropdown-item command="1" label="Item 1"></my-dropdown-item>
          <my-dropdown-item command="2" label="Item 2" />
          <my-dropdown-item command="3" disabled>Item 3</my-dropdown-item>
          <my-dropdown-item command="4" divided>Item 4</my-dropdown-item>
        </template>
      </my-dropdown>
    </div>
    <div class="col">
      <div class="desc">通过 props 传入 item</div>
      <my-dropdown :items="items">
        <span class="dropdown-link">
          Dropdown List
          <my-icon icon="angle-down" />
        </span>
      </my-dropdown>
    </div>
  </div>
</template>

<style scoped>
.row {
  display: flex;
  flex-wrap: wrap;
}
.col {
  flex: 1;
}
.desc {
  display: block;
  color: var(--er-text-color-secondary);
  font-size: 14px;
  margin-bottom: 20px;
}
.dropdown-link {
  display: flex;
  align-items: center;
  cursor: pointer;
  color: var(--er-color-primary);
  i {
    margin-left: 8px;
  }
}
</style>

点击菜单触发指令事件 ​

可以通过 command 事件来设置点击 item 后的事件。

command

<script setup lang="ts">
import {
  type DropdownItemProps,
  type DropdownCommand,
  MyMessage,
} from "@qiwen72/my-component";

const items: DropdownItemProps[] = [
  { command: "1", label: "Item 1" },
  { command: "2", label: "Item 2" },
  { command: "3", label: "Item 3", disabled: true },
  { command: "4", label: "Item 4", divided: true },
];

function handleClick(command: DropdownCommand) {
  MyMessage.success("click on item " + command);
}
</script>

<template>

    <div class="col">

          <span>command</span>
        </div>
      </my-dropdown>
    </div>
  </div>
</template>

下拉方法 ​

可以获取到 dropdown 实例的 open 和 close 方法，在其他地方进行触发

command

<script setup lang="ts">
import {
  type DropdownItemProps,
  type DropdownInstance,
} from "@qiwen72/my-component";
import { ref } from "vue";

const _myDropDown = ref<DropdownInstance>();
const items: DropdownItemProps[] = [
  { command: "1", label: "Item 1" },
  { command: "2", label: "Item 2" },
  { command: "3", label: "Item 3", disabled: true },
  { command: "4", label: "Item 4", divided: true },
];
</script>

<template>
  <div class="row">
    <div class="col">
      <my-button @click="() => _myDropDown?.open()">Open</my-button>
      <my-button @click="() => _myDropDown?.close()">Close</my-button>
      <div class="desc">command</div>
      <my-dropdown :items="items" ref="_myDropDown">
        <div class="trigger">
          <span>command</span>
        </div>
      </my-dropdown>
    </div>
  </div>
</template>

<style>
.row {
  margin-right: 5px;

}
.col {
  flex: 1;
}
.desc {
  margin-bottom: 10px;
  color: var(--er-text-color-secondary);
}
.trigger {
  display: flex;
  span {
    align-items: center;
    color: var(--er-color-primary);
  }
}
</style>

触发对象样式 ​

设置 split-button 属性来让触发下拉元素呈现为按钮组，按钮可以有自己单独的功能

<script setup lang="ts">
import { type DropdownItemProps, MyMessage } from "@qiwen72/my-component";

const items: DropdownItemProps[] = [
  { command: "1", label: "Action 1" },
  { command: "2", label: "Action 2" },
  { command: "3", label: "Action 3", disabled: true },
  { command: "4", label: "Action 4", divided: true },
];

function handleClick() {
  MyMessage.info("button click");
}
</script>

<template>
  <div class="row">
    <my-dropdown :items="items">
      <my-button type="primary">
        Dropdown List
        <my-icon icon="angle-down" style="margin-left: 8px" />
      </my-button>
    </my-dropdown>
    <my-dropdown
      :items="items"
      type="primary"
      @click="handleClick"
      split-button
    >
      Dropdown List
    </my-dropdown>
  </div>
</template>

<style scoped>
.row div {
  margin-right: 16px;
}
</style>

dropdown API ​

Props ​

NameDescriptionTypeDefault
type菜单按钮类型与 button 类型相同
size大小large default smalldefault
items下拉 itemsDropdownItemProps[]
hide-on-click点击菜单项时是否隐藏下拉菜单booleantrue
split-button下拉按钮与功能按钮一套booleanfalse
show-timeout出现缓冲时间number0
hide-timeout隐藏缓冲时间number200

Events ​

NameDescriptionType
visible-change气泡显示(value: boolean)=> void
clicksplit-buttton 为 true 时,点击左侧按钮触发(event: MouseEvent) => void
command点击菜单项时触发(value: DropdownCommand) => void

NameDescription
